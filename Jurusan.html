JURUSAN SEKOLAH

  1. Jurusan
12 JURUSAN SMK NEGERI 1 DENPASAR TERLENGKAP

1. Teknik Konstruksi dan Perumahan (TKP)

Kompetensi:
  1. Pembacaan dan Penafsiran Gambar Teknik
  2. Metode Konstruksi
  3. Estimasi Biaya dan Perencanaan Proyek.
  4. Praktek Kerja Konstruksi.
  5. Teknik Sipil dan Infrastruktur.
Prospek Kerja:
  1. Teknisi Konstruksi Estimator Proyek
  2. Asisten Insinyur
  3. Teknisi Instalasi
  4. Pengusaha Konstruksi
  5. Office Administration

2. Desain Pemodelan dan Informasi Bangunan (DPIB)

Kompetensi:
  1. Dasar-dasar gambar teknik.
  2. Menggambar dengan perangkat lunak (komputer)
  3. Menggambar konstruksi gedung
  4. Menghitung anggaran biaya bangunan
  5. Teknik pengukuran tanah.
  6. Branding dan Corporate Identity
Prospek Kerja:
  1. Drafter
  2. Kontraktor / Pemborong
  3. Quality Control
  4. Konsultan Perencana
  5. Quantity Surveyor

3. Teknik Audio Video (TAV)

Kompetensi:
  1. Merencanakan sistem / peralatan elektronika berbasis analog dan mikrokontroler
  2. Merakit dan membuat sistem / peralatan elektronika berbasis analog maupun mikrokontroler
  3. berbasis analog maupun mikrokontroler
  4. Microcontroller programming
Prospek Kerja:
  1. Teknisi Elektronika Umum
  2. Teknisi Elektronika Medik
  3. Desainer & Programmer Hardware Elektronika
  4. Desainer & Programmer Sistem Elektronika Otomatis

4. Teknik Pendingin dan Tata Udara (TPTUP)

Kompetensi:
  1. Pemanasan seperti Heat Pump dan Water Heater
  2. Tata Udara dan Pendinginan seperti Air Conditioning (AC) dan Refrigerator
Prospek Kerja:
  1. Engineering di bidang HVAC (Heating, Ventilation, and Air Conditioning)
  2. Instalatur HVAC
  3. Bekerja di Industri HVAC

5. Pemeliharaan Mesin (PM)

Kompetensi:
  1. Mengoperasikan mesin-mesin produksi manual maupun mesin CNC, Bubut, Frais, Sekrap, Bor, dan Gerinda.
  2. Menggambar manual maupun dengan CAD.
  3. Pengenalan dan pengolahan bahan, perlakuan panas, mengukur dengan alat ukur mekanik, menggunakan perkakas tangan.
Prospek Kerja:
  1. Industri manufakturing (operator mesin)
  2. Programer (programer dan operator mesin bubut CNC, programer operator mesin frais CNC)
  3. Entrepreneur dalam bidang pemesinan

6. Teknik Sepeda Motor (TSM)

Kompetensi:
  1. Teknisi atau Mekanik pada Bengkel Sepeda Motor.
  2. Teknisi perakitan atau teknisi produksi pada industri Sepeda Motor.
  3. Wirausahawan dalam bidang Otomotif (Bengkel Sepeda Motor).
Prospek Kerja:
  1. Teknisi atau Mekanik pada Bengkel Sepeda Motor.
  2. Teknisi perakitan atau teknisi produksi pada industri Sepeda Motor.
  3. Wirausahawan dalam bidang Otomotif (Bengkel Sepeda Motor)

7. Teknik Kendaraan Ringan (TKR)

Kompetensi:
  1. Perawatan dan perbaikan mesin, kelistrikan, serta chasis kendaraan.
  2. Menggunakan alat ukur dan alat diagnostik otomotif.
  3. Mengoperasikan peralatan bengkel dan memahami sistem injeksi serta ECU
Prospek Kerja:
  1. Mekanik atau teknisi otomotif.
  2. Service advisor atau quality control.
  3. Wirausaha (bengkel atau bisnis suku cadang).

8. Teknik Instalasi Tenaga Listrik (TITL)

Kompetensi:
  1. Dasar dan pengukuran listrik
  2. Pekerjaan dasar elektromekanik
  3. Instalasi penerangan listrik
  4. Instalasi tenaga listrik
  5. Instalasi motor listrik
Prospek Kerja:
  1. Electrician
  2. Teknisi Sistem Energi Terbarukan
  3. Estimator Proyek Listrik
  4. Teknisi Keamanan Listrik
  5. Teknisi Listrik

9. Teknik Komputer Jaringan (TKJ)

Kompetensi:
  1. Menginstal perangkat komputer personal dan meng-install sistem operasi serta aplikasi.
  2. Menginstal perangkat jaringan berbasis lokal.
  3. Menginstal perangkat jaringan berbasis luas.
Prospek Kerja:
  1. Kantor-kantor pemerintah, sekolah, dan lainnya.
  2. TSS (Technical Support System).
  3. ISP (Internet Service Provider).
  4. Instalasi jaringan komputer.

10. Rekayasa Perangkat Lunak (RPL)

Kompetensi:
  1. Teknik Dasar Operasi dan Perawatan Komputer
  2. Mengoperasikan Software Office (Perkantoran)
  3. Mengoperasikan Visual Programming
  4. Mengoperasikan Object Oriented Programming (OOP)
  5. Mengoperasikan Database Programming
  6. Mengoperasikan Web Programming
  7. Mengoperasikan Advance Programming
Prospek Kerja:
  1. Web Development
  2. Game Development
  3. Application Development
  4. IT Consultant
  5. Office Administration

11. Desain Komunikasi Visual (DKV)

Kompetensi:
  1. Penulisan Naskah
  2. Sinematografi
  3. Penyuntingan
  4. Desain Suara
  5. Produksi Film
Prospek Kerja:
  1. Sutradara
  2. Produser
  3. Sinematografer
  4. Penyunting Film
  5. Manajer Produksi

12. Produksi Perfilman (PRF)

Kompetensi:
  1. Teknik Dasar Operasi dan Perawatan Komputer
  2. Mengoperasikan Software Office (Perkantoran)
  3. Mengoperasikan Visual Programming
  4. Mengoperasikan Object Oriented Programming (OOP)
  5. Mengoperasikan Database Programming
  6. Mengoperasikan Web Programming
  7. Mengoperasikan Advance Programming
Prospek Kerja:
  1. Web Development
  2. Game Development
  3. Application Development
  4. IT Consultant
  5. Office Administration
